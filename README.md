# Практическая работа №1: CSS-препроцессоры

## Кратко
В проекте использован **SASS (SCSS)** для стилизации карточек товаров.

## Что сделано
- Реализована карточка товара (название, описание, фото).
- Использованы переменные (цвета, шрифты, радиусы).
- Использован миксин для базовых стилей карточки.
- Применена вложенность селекторов (`&__image`, `&__title`, `&__description`, `&__btn`).
- Получен итоговый CSS после компиляции.

## Файлы в проекте
- Исходные стили SCSS: [styles/main.scss](styles/main.scss)
- Скомпилированные стили CSS: [styles/main.css](styles/main.css)
- Дополнительно SCSS в клиенте: [client/src/pages/ProductsPage/ProductsPage.scss](client/src/pages/ProductsPage/ProductsPage.scss)

## Как запустить
1. Установить зависимости: `npm install`
2. Скомпилировать SCSS в CSS: `npx sass styles/main.scss styles/main.css`
3. Запустить проект: `npm start`

# Практическая работа №2: Сервер на Node.js + Express

## Кратко
Реализован backend API на Express для управления товарами.

## Что сделано
- Настроен сервер на Node.js + Express.
- Подключены middleware: `express.json()`, `express.urlencoded()`, `express.static()`.
- Реализованы CRUD-операции для товаров:
	- `GET /api/products` — получить все товары;
	- `GET /api/products/:id` — получить товар по id;
	- `POST /api/products` — добавить товар;
	- `PATCH /api/products/:id` — обновить товар;
	- `DELETE /api/products/:id` — удалить товар.
- Добавлена базовая валидация входных данных и обработка `404`/`500`.

## Файлы в проекте
- Сервер и маршруты API: [app.js](app.js)
- Настройки зависимостей и скриптов: [package.json](package.json)

## Как запустить
1. Установить зависимости: `npm install`
2. Запустить сервер: `npm start`
3. Проверка API: `http://localhost:3000/api/products`

# Практическая работа №4: API + React

## Кратко
Связан backend на Express и frontend на React в единый сайт интернет-магазина.

## Что сделано
- На сервере реализован CRUD API для товаров (`/api/products`) и хранение списка из 10+ товаров.
- На клиенте реализованы карточки товаров и операции: просмотр, создание, редактирование, удаление.
- Для запросов к API подключен `axios` и вынесен отдельный API-слой.
- Настроен CORS для корректной работы клиента и сервера на разных портах.

## Файлы в проекте
- Backend API: [app.js](app.js)
- Клиентский API: [client/src/api/index.js](client/src/api/index.js)
- Страница каталога: [client/src/pages/ProductsPage/ProductsPage.jsx](client/src/pages/ProductsPage/ProductsPage.jsx)
- Компоненты карточек и модального окна: [client/src/components](client/src/components)

## Как запустить
1. Установить зависимости в корне: `npm install`
2. Установить зависимости клиента: `cd client && npm install`
3. Запустить backend: `npm start`
4. Запустить frontend: `cd client && npm run dev`
