# Практическая работа №2: Сервер на Node.js + Express

## Кратко
Реализован backend API на Express для управления товарами.

## Что сделано
- Настроен сервер на Node.js + Express
- Подключены middleware: `express.json()`, `express.urlencoded()`, `express.static()`
- Реализованы CRUD-операции для товаров:
	- `GET /api/products` — получить все товары;
	- `GET /api/products/:id` — получить товар по id;
	- `POST /api/products` — добавить товар;
	- `PATCH /api/products/:id` — обновить товар;
	- `DELETE /api/products/:id` — удалить товар.
- Добавлена базовая валидация входных данных и обработка `404`/`500`

# Практическая работа №4: API + React

## Кратко
Связан backend на Express и frontend на React в единый сайт интернет-магазина.

## Что сделано
- На сервере реализован CRUD API для товаров (`/api/products`) и хранение списка из 10+ товаров
